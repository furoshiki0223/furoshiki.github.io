Hiroshi Kawada(ja:川田 寛)

Hiroshi Kawada is a Web developer at NTT Comware Corporation in Japan and very interested in Web performance, mobile UX, so he publishes some articles. He has joined in Microsoft MVP for Internet Explorer, html5j and Japan Cordova UG(co-organizer) to make the open technologies more progressive.

川田寛は、NTTコムウェア所属の日本のWeb開発者。Webパフォーマンス、モバイルのUXに対して高い関心を抱き、エンジニア業の傍ら、雑誌やオンラインメディアへいくつかの記事を寄稿しています。また、Microsoft MVP(Internet Explorer)受賞、Web技術者コミュニティ「html5j」の運営や、日本Cordovaユーザー会の立ち上げと運営に関わり、オープンな技術の普及・推進に貢献しています。
