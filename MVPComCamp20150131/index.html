What is HTML5？

W3C勧告になったHTML5

Abstract

This specification defines the 5th major revision of the core language of the World Wide Web: the Hypertext Markup Language (HTML). In this version, new features are introduced to help Web application authors, new elements are introduced based on research into prevailing authoring practices, and special attention has been given to defining clear conformance criteria for user agents in an effort to improve interoperability.

ハイパーテキストマークアップ言語（HTML）：この仕様は、World Wide Webのコア言語の5番目の改訂を定義する。このバージョンで、新しい機能は、Webアプリケーションの作成者を支援する。また、新しい要素は、一般的なオーサリングプラクティスの研究に基づいて導入される。これらは、ユーザーエージェントの相互運用性を向上させるため、明確な適合基準として定義する。

で、中身は

HTML5という仕様

  • Web技術全体で共通する概念の定義
  • タグ・メタタグ・formタグ
  • HTML言語仕様とブラウザの仕組み
  • レンダリングの仕様
  • 廃止するタグの扱い

かっこいいグラフィクスを扱う機能

Canvas, SVG, CSS

最近は『WebGL』。4月にWindows Phone 8.1、9月にAndroid Browser 37、同月にiOS 8と、2014年にモバイルで一気に動くようになった！FirefoxOSは2013年の段階で既に動作。

→ 今回のHTML5の仕様には含まれない

高度な音や映像を扱う機能

Audioタグ。ゲームやサウンド制作のためにも『Web Audio』や『Web MIDI API』の実装が進んで欲しいが、IE TPでも未実装。

Videoタグ。MSE、VP9、EME等の実装が進み、長年FlashだったYoutubeがようやくVideoタグの利用に踏み切る。

→ 今回のHTML5の仕様には含まれない

いろんな事ができる通信の機能

WebSocket, HTTP2.0

最近は『WebRTC』がアツい。長年Skypeの実装絡みで採用が難しかったが、『ORTC』がIEに実装されることになり、やっと実用的に！

→ 今回のHTML5の仕様には含まれない

ストレージとかオフラインとか…

WebStorage。最近は『IndexedDB』がアツい。直前はiOS8のWK WebViewのみにサポートされるという噂だったけど、どうやらSafariでも動くっぽい。

AppCache。これからは『ServiceWorkers』。Chromeも昨年末から動くように！

→ 今回のHTML5の仕様には含まれない
